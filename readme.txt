== Description ==
 
Change your category name, tag name or taxonomy name.

**Major features of this plugin include**
* Add text before or after current post title
* Select category, tag or taxonomy to change

 
== Installation ==
 
The plugin is simple to install:
 
1. Download change-category-name.zip
1. Upload `change-category-name` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
